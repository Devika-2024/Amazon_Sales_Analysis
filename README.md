# Amazon Sales Analysis Project

## Project Overview
This project analyzes Amazon sales data to uncover trends in revenue, product categories, order status, fulfillment methods, and regional performance.

The goal of this project is to demonstrate real-world data analysis skills using Python, Pandas, Matplotlib, and Seaborn.

---

## Tools & Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook
- Excel
- GitHub

---

## Dataset
Amazon sales transactional dataset containing:
- Order ID
- Order Date
- Product Category
- Quantity
- Revenue
- Order Status
- State / City
- Fulfillment Type

---

## Data Cleaning Performed
- Removed unnecessary columns
- Fixed date formats
- Handled missing values
- Converted data types
- Created Month / Year columns
- Prepared cleaned dataset for analysis

---

## Key Business Insights
1. Monthly sales trends identified peak revenue periods.
2. Top product categories contributed most revenue.
3. Certain states generated higher order volume.
4. Majority of orders were successfully shipped.
5. Amazon fulfillment handled a large portion of orders.
